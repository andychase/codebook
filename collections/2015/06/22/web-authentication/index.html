Web Authentication

Jun 22, 2015 • Andy Chase

Authentication aka User/Passwords/Logins

Authentication is an important subject. Civilization centers around people, and on the internet the concept of a person having individuality and control over their own actions need to be emulated.

Authentication is a bit tricky on the web because the web is decentralized and stateless, but several solutions have been invented and more arrive all the time. This collection hopes to introduce some important thoughts related to authentication on the web.

Overview: Different ways to do authentication

Username/passwords are the cultural norm of web-based authentication, but other methods exist.

Read this if you want to learn how to (properly) set up login/register based authentication so your site doesn't instantly get hacked once its online.
